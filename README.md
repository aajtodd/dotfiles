# Quickstart

Install stow

Mac: `brew install stow`

See https://alexpearce.me/2016/02/managing-dotfiles-with-stow/

```
$ git clone git@github.com:aajtodd/dotfiles.git ~/.dotfiles
$ cd ~/.dotfiles
$ stow nvim zsh # plus whatever else you'd like
```
