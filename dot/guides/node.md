# node — fnm version manager

Loaded with `--use-on-cd`: entering a dir with `.nvmrc`/`.node-version`
auto-switches. node/npm present in every shell.

```
fnm list                 installed versions
fnm install --lts        install latest LTS
fnm use 22               switch this shell to node 22
fnm default lts-latest   default for new shells
```
